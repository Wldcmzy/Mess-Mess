无法正常游玩游戏？

    若聊天框提示形如这样的信息, 请调整lerp:

    <Lerp> wldcmzy was moved to spectators for lerp 500.0

    <Lerp> Illegal lerp value (min: 0.0, max: 67.0)

    方法为:

    按 ` 建打开控制台(前提是已经在设置中启用控制台), 输入以下内容:

    cl_interp_ratio 0; cl_interp 0

常用命令

    !match 加载插件 !rmatch 解除当前插件

    !mvp 查询战绩 !bonus 查询分数 !tank 查询坦克位置

    !tankhud 显示或关闭tank信息UI(生还不可用)

    !voteboss 投票设置坦克位置

    !s 旁观 !sepchud 打开观战者UI

    !pause 暂停 !unpause 解除暂停

    !slots x 投票修改房间人数上限为x

    对局开始前: !hide 隐藏菜单 !show 显示菜单
